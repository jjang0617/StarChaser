# ⭐ StarChaser (스타체이서)
> "오늘 밤, 별 보러 가도 될까?" - 실시간 기상 데이터 기반 별 관측 지수 서비스

## 🌌 Core Tech Stack
- **Frontend:** React Native (Expo), NativeWind
- **Backend:** NestJS, Supabase (PostgreSQL + PostGIS)
- **Data:** 기상청 단기예보 API, 에어코리아, KASI

## 🚀 Getting Started
1. `git clone https://github.com/계정/starchaser.git`
2. `npm install`
3. `.env` 파일 설정 (팀 채널 공지 확인)
4. `npx expo start`
